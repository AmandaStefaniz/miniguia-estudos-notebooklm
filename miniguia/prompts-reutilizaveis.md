# Prompts Reutilizáveis

## Para revisar o tema

Explique Maquiavel em linguagem simples, destacando contexto histórico, principais conceitos e relevância atual.

## Para estudar conceitos

Explique os conceitos de virtù, fortuna, realismo político e necessidade política em Maquiavel.

## Para comparar obras

Compare O Príncipe e Os Discursos, destacando diferenças de contexto, objetivo e visão política.

## Para criar exemplos atuais

Relacione os conceitos de Maquiavel com situações contemporâneas de liderança, política, empresas ou redes sociais.

## Para criar questões de estudo

Crie 10 perguntas sobre Maquiavel, com respostas comentadas, para revisão antes de uma apresentação.

## Para criar resumo final

Crie um resumo estruturado sobre Maquiavel, separando: contexto histórico, conceitos principais, obras, interpretações e atualidade.

## Para aprofundar criticamente

Explique por que Maquiavel é frequentemente mal interpretado e quais são as principais leituras possíveis sobre sua obra.
